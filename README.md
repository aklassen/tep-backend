# TEP - Backend - (yet another) Tiny Event Planner
[![Build Status](https://travis-ci.org/aklassen/tep-backend.svg?branch=master)](https://travis-ci.org/aklassen/tep-backend)

## Description

TEP is using the [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
